Browsable Multivalue SuggestBox Using REST

Enter different Crayola crayon colors in the text box either by

  • typing a few letters at a time and selecting from the suggestions popup
  • or by copying and pasting using the keyboard (Ctrl-C and Ctrl-V) the following string including the last trailing comma: carn, banana, red, blue green, almond, yellow orange,
  • or by typing '*' to browse
  • or using any combination of the above.
Then click the 'Show Values' buttons to see the Hex codes for the colors selected.
